hexo搭建博客详解

博客搭建

1，安装node
2，安装博客所需插件
    npm install -g hexo-cli
3，初始化一个博客
    hexo init
4，启动博客
    hexo s
5，新建博客
    hexo n "我的第一篇博客"
6，进入新建好的博客下面(命令行编写博客内容)
    cd source/_posts/
7，回退到blog文件夹
    cd ../../
8，编辑器打开编写博客内容
9，清除
    hexo clean
10，hexo g

博客部署到github上

新建一个仓库，仓库名称要是xxxgithub.io
1，部署到github上
    npm install --save hexo-deplpyer-git
2，安装完成改下配置_config.yml
        deploy:
            type: git
            repo: https://github.com/xxx/xx.github.io.git
            branch: master

刷新gitHub看到东西更改了（部署成功）
访问通过   xxx.github.io

博客主题更改

https://github.com/litten/hexo-theme-yilia
1，克隆到本地
git clone https://github.com/litten/hexo-theme-yilia.git theme/yilia
2，theme: yilia
3，推到远端
hexo d
